# Design Brief: 脱口秀个人海报

## 规格

- 类型: standup-poster
- 尺寸: 1080×1920px（竖版 9:16）
- 输出: PNG
- 用途: 抖音/B站视频封面

## 信息层级

1. **主视觉**: {{actor_name}}（最大字重，视觉锚点）
2. **副标题**: {{title}}（段子名或主题）
3. **Hook**: {{hook_line}}（一句话金句，若有）
4. **辅助信息**: {{date}} | {{venue}}（若有演出信息）
5. **底部**: Cclaw 喜剧龙虾（可选品牌标识）

## 风格指引

- 喜剧视觉语言：强对比、大胆用色、冲击力
- 字体：粗体无衬线，力量感
- 情绪基调：{{mood}}（如：讽刺/温暖/荒诞/治愈/炸裂）
- 参考：波兰海报学派 / 波普艺术 / 日系综艺字幕风格
- 文字占比：10% 视觉 + 90% 设计（极简文字，让设计说话）
