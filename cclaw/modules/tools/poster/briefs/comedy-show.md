# Design Brief: 演出宣传海报

## 规格

- 类型: comedy-show
- 尺寸: 1920×1080px（横版 16:9）
- 输出: PNG
- 用途: 线下演出宣传、朋友圈海报

## 信息层级

1. **主视觉**: {{show_name}}（演出名称，最大字重）
2. **时间**: {{date}} {{time}}
3. **地点**: {{venue}}
4. **阵容**: {{actors}}（演员名单）
5. **Slogan**: {{slogan}}（一句话宣传语）
6. **辅助**: {{price}} | {{ticket_info}}（票价/购票信息）

## 风格指引

- 演出感：海报要有事件感、聚光灯感
- 强调时间紧迫性（倒计时感）和稀缺性
- 字体：粗体标题 + 清晰正文，信息层次分明
- 情绪基调：热闹/期待/隆重
- 参考：音乐节海报 / 脱口秀俱乐部海报 / 演唱会宣传
- 文字占比：信息类海报可适当增加文字密度，但保持视觉优先
