# 配方：段子金句卡

> 从脱口秀/漫才/小品中提取金句，生成社交分享卡。

## 触发条件

用户说以下任意一种：
- 金句卡 / 分享卡 / 朋友圈配图
- 把这段做成图 / 帮我生成一张图
- 从段子中提取一句做成海报

## 快速流程

1. 提取金句（从创作输出中选取最炸的一句，或用户指定）
2. 填充 `briefs/social-card.md` 模板
3. 调用 canvas-design 生成 1200×630 PNG

## 选句技巧

- 优先选有完整反转的句子（铺垫+抖包袱一体）
- 优先选自嘲/共鸣类（传播力强）
- 控制字数在 40 字以内（社交卡片场景）
- 如果段子太长，提取最精华的转折句

## 示例

输入：「35岁，一个Bug都没修好的人生」脱口秀

推荐金句：
「我35岁了，修了十年Bug，最大的成就就是——证明了一个Bug可以活十年。」

→ 填充 brief → 调用 canvas-design → 输出 1200×630 PNG
