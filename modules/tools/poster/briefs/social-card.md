# Design Brief: 社交媒体分享卡

## 规格

- 类型: social-card
- 尺寸: 1200×630px（横版 1.91:1）
- 输出: PNG
- 用途: 微信朋友圈 / 微博 / 社交平台分享

## 信息层级

1. **主内容**: {{quote}}（金句/段子原文，核心视觉元素）
2. **来源**: {{source}}（出处，若有）
3. **作者**: {{author}}（作者名，若有）
4. **品牌**: Cclaw 喜剧龙虾（可选）

## 风格指引

- 金句卡=文字即视觉：文字排版本身构成设计
- 强调文字的力量感和节奏感
- 背景简洁或有微妙纹理，不抢文字风头
- 字体：衬线/无衬线混搭增加层次
- 情绪基调：{{mood}}（与段子情绪一致）
- 参考：书中金句卡 / 黑色幽默 meme / 日系俳句卡片
- 文字占比：可适当提高至 30-40%（金句卡以文字为核心元素）
