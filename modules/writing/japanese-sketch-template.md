# 日式短剧（コント）输出模板

> 适用：5分钟极快节奏短剧。核心是"极端性格 + 明确目标 + 阻碍升级"。

## ⚡ 死磕6条铁律（每次创作前必看）

1. **角色永远第一**：性格贯穿全程，不崩人设
2. **目标明确**：每个角色都有一个要完成的事
3. **起承转合闭环**：5分钟也要有完整故事线
4. **认真的荒诞**：角色越认真，观众越想笑
5. **节奏铁律**：15秒小笑点，1分钟大爆点
6. **装傻+吐槽**：用行为装傻，用反应吐槽

## 输出规格

### 结构规范

```
【日式短剧标题】

### 舞台布景
（场景描述，1-2句话）

### 道具
（列出关键道具）

### 灯光
（开场→转折→结尾的光线变化）

### 音效
（关键音效节点）

---

**【开场】**（30秒）
建立人物 + 场景，让观众知道谁、在哪、想干嘛

**【发展】**（1分钟）
第一次冲突——角色的目标受阻

**【转折】**（2分钟）
冲突升级、叠加新矛盾、人物被逼到极限
- 阻碍1
- 阻碍2（叠加）
- 阻碍3（彻底失控）

**【收尾】**（30秒）
反转 / 荒诞收尾（必须有明确结局）

---

**【笑点统计】**
共 X 个笑点，平均每分钟 X.X 个
```

## 创作步骤

### Step 1：设计角色（最关键）

选择一个极端性格：
- **偏执**：对某事有病态执着
- **懦弱**：怕这怕那，不敢行动
- **自大**：过度自信，轻视一切
- **较真**：对细节有病态要求

**人设铁律**：一旦确定，从头到尾不能变。所有笑点来自性格驱动行为。

### Step 2：设定情境

选择日常场景：办公室、医院、便利店、家里、学校、餐厅

问自己：如果把这个极端性格放进这个日常场景，会发生什么？

### Step 3：确定目标

角色想要什么？（只能有一个核心目标）
- 请假
- 看病
- 要钱
- 证明自己
- 买某个东西
- 说服某人

### Step 4：设计阻碍升级

```
阻碍1：初步受阻
阻碍2：叠加新矛盾（不能重复阻碍1的梗）
阻碍3：彻底失控（所有矛盾同时爆发）
```

每次阻碍都要让情况更糟，角色越努力越糟糕。

### Step 5：设计收尾

选择一种：
- **反转**：观众以为是A，结果是B
- **荒诞**：完全不讲道理的结局
- **认命**：角色接受失败，但以极端方式

## 手法清单（日式短剧优先）

| 手法 | 说明 |
|------|------|
| 装傻 & 吐槽 | 角色行为装傻 + 另一个角色反应吐槽 |
| 15秒法则 | 每15秒必须有一个笑点 |
| 认真演傻 | 哪怕演傻子，也要一本正经地傻 |
| 目标升级 | 角色越努力，情况越糟 |
| 性格驱动 | 笑点来自性格，不是硬加梗 |

## 笑点密度要求

```
最低标准：≥4个笑点/分钟（5分钟 ≈ 20个笑点）
优秀标准：≥5个笑点/分钟（5分钟 ≈ 25个笑点）
顶级标准：≥6个笑点/分钟（5分钟 ≈ 30个笑点）
```

## 三人结构模板

如果需要三人短剧：

```
角色A：主装傻（极端性格持有者）
角色B：主吐槽（正常人视角，对A的行为做出反应）
角色C：旁观者/搅局者（强化冲突，可以是无辜受害者或另一个极端性格）
```

## 格式规范

每个笑点用 `**【笑点N】**` 标注，后面简短说明笑点来源（性格驱动/动作/台词/反应）
