# Cclaw v1.9.0 风格体系

## 任务目标
在每个喜剧大类（脱口秀/漫才/Sketch/日式短剧）下增加"风格"作为第二层维度，支持案例风格标注、AI推荐风格、创作时自动推断。

## 关键决策
1. **风格 ≠ 题材** — 风格是"怎么讲"的调性，同一话题可以用不同风格
2. **风格插入时机** — 四问之后第五步（可跳过），或情况A自动推断
3. **案例带标签** — 每个案例顶部 `style:` frontmatter 标记风格
4. **按风格筛选案例** — 创作时优先读匹配风格的案例
