# Cclaw 命令配置

> 本文件记录 Cclaw 技能支持的所有命令模式。

## 文本创作命令

| 命令关键词 | 输出类型 | 优先级 |
|-----------|---------|--------|
| 脱口秀 / standup / 单口喜剧 | 脱口秀 | 高 |
| 小品 / sketch / 情景喜剧 / 素描喜剧 | 小品剧本 | 高 |
| 漫才 / manzai / 日本漫才 | 漫才对口 | 高 |
| 剧本 / 喜剧剧本 / 独幕剧 | 完整剧本 | 高 |
| 讽刺 / satire / 讽刺文章 | 讽刺文 | 中 |
| 仿讽 / 戏仿 / 恶搞 / parody | 仿讽作品 | 中 |
| 荒诞 / 荒诞剧 / 荒诞喜剧 | 荒诞剧 | 中 |
| 段子 / 笑话 / joke / 搞笑段子 | 独立笑点 | 中 |
| 改编 / 改成喜剧 / 喜剧化 | 原文喜剧改编 | 高 |

## 视频工具命令

| 命令关键词 | 输出类型 | 说明 |
|-----------|---------|------|
| 剪辑 / cut / 编辑视频 | FFmpeg 剪辑命令 | 解析自然语言脚本，生成剪辑命令 |
| 拼接 / concat / 合并视频 | FFmpeg 拼接命令 | 多段视频合并 |
| 删 / delete / 删除片段 | FFmpeg 剪切命令 | 删除指定时间段 |
| 提取 / extract / 截取片段 | FFmpeg 截取命令 | 提取指定时间段 |
| 加音乐 / BGM / 加背景音乐 | FFmpeg 混音命令 | 混音合成 |
| 加字幕 / subtitle | FFmpeg 字幕命令 | 烧入字幕 |
| 转码 / encode / 转换格式 | FFmpeg 转码命令 | 格式转换 |
| 查看信息 / info / mediainfo | ffprobe 分析命令 | 输出视频信息 |

## 默认命令

无指定类型时 → 优先尝试文本创作，或询问用户。

## 命令扩展规则

新增命令格式：
```
| 关键词 / 同义词 | 输出类型 | 说明 |
```

扩展时直接追加行。
